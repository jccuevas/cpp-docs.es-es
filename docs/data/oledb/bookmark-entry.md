# BOOKMARK_ENTRY
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Enlaza la columna de marcador.  
  
## Sintaxis  
  
```  
  
BOOKMARK_ENTRY(  
variable  
 )  
  
```  
  
#### Parámetros  
 *variable*  
 \[in\] La variable que se enlazará a la columna de marcador.
