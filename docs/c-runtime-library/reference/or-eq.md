# <a name="oreq"></a>or_eq

Alternativa al operador &#124;=.

## <a name="syntax"></a>Sintaxis

```C

#define or_eq |=

```

## <a name="remarks"></a>Comentarios

La macro produce el operador &#124;=.

## <a name="example"></a>Ejemplo

```cpp
// iso646_oreq.cpp
// compile with: /EHsc
#include <iostream>
#include <iso646.h>

int main( )
{
   using namespace std;
   int a = 3, b = 2, result;

   result= a |= b;
   cout << result << endl;

   result= a or_eq b;
   cout << result << endl;
}
```

```Output
3
3
```
