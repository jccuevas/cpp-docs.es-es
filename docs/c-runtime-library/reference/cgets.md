# <a name="cgets"></a>cgets
Esta función POSIX está en desuso. En su lugar, use la función [_cgets](../../c-runtime-library/cgets-cgetws.md), compatible con ISO C++, o la función [_cgets_s](../../c-runtime-library/reference/cgets-s-cgetws-s.md) con mejoras de seguridad.  
  
> [!IMPORTANT]
>  Esta API no se puede usar en aplicaciones que se ejecutan en Windows en tiempo de ejecución.
