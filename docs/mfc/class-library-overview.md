# <a name="class-library-overview"></a>Información general de la biblioteca de clases

En esta información general se clasifican y describen las clases de la versión 9,0 de biblioteca MFC (MFC). Las clases de MFC, tomadas juntas, constituyen un marco de trabajo de la aplicación, el marco de una aplicación escrita para la API de Windows. La tarea de programación consiste en rellenar el código específico de la aplicación.

Para obtener información general sobre el marco de trabajo, vea [usar las clases para escribir aplicaciones para Windows](using-the-classes-to-write-applications-for-windows.md). Algunas de las clases enumeradas anteriormente son clases de uso general que se pueden usar fuera del marco de trabajo y proporcionan abstracciones útiles como colecciones, excepciones, archivos y cadenas.

Para ver la herencia de una clase, use el [gráfico de jerarquía de clases](hierarchy-chart.md).

Además de las clases enumeradas en esta información general, la biblioteca MFC contiene una serie de funciones globales, variables globales y macros. Encontrará información general y una lista detallada de estos temas en el tema [macros y globales de MFC](reference/mfc-macros-and-globals.md), que siguen la referencia alfabética a las clases MFC.
