# Advertencia del compilador (nivel 4) C4152
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

extensión no estándar; conversión de puntero a función\/datos en la expresión  
  
 Un puntero a función se ha convertido a un puntero a datos o desde este. Esta conversión se permite en las extensiones de Microsoft \(\/Ze\), pero no en ANSI C.
