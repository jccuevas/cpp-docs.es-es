# <a name="compiler-warning-level-1-c4502"></a>Advertencia del compilador (nivel 1) C4502

'especificación de vinculación' requiere el uso de la palabra clave 'extern' y debe preceder al resto de los especificadores

Se especificó una vinculación sin la `extern` palabra clave. Vinculación no es relevante para los tipos no extern.

El compilador supone que el `extern` palabra clave.
