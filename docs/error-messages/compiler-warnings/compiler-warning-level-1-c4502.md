# <a name="compiler-warning-level-1-c4502"></a>Advertencia del compilador (nivel 1) C4502

' especificación de vinculación ' requiere el uso de la palabra clave ' extern ' y debe preceder a todos los demás especificadores

Se especificó una vinculación sin la palabra clave `extern`. La vinculación no es relevante para los tipos no extern.

El compilador asumió la palabra clave `extern`.
