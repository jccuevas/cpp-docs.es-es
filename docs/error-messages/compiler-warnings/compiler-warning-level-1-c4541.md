# Advertencia del compilador (nivel 1) C4541
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'identificador' se ha utilizado en el tipo polimórfico 'tipo' con \/GR\-; puede tener como resultado un comportamiento impredecible  
  
 Se intentó usar una característica que requiere información en tiempo de ejecución sin habilitar la información de tipos en tiempo de ejecución.  Vuelva a compilar con la opción [\/GR](../../build/reference/gr-enable-run-time-type-information.md).
