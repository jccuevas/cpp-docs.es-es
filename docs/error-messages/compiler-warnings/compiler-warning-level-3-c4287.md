# <a name="compiler-warning-level-3-c4287"></a>Advertencia del compilador (nivel 3) C4287

' operador ': no coinciden las constantes sin signo o negativas

Se usó una variable sin signo en una operación con un número negativo.

De forma predeterminada, esta advertencia está desactivada. Vea [Advertencias del compilador desactivadas de forma predeterminada](../../preprocessor/compiler-warnings-that-are-off-by-default.md) para más información.

## <a name="example"></a>Ejemplo

En el ejemplo siguiente se genera C4287:

```cpp
// C4287.cpp
// compile with: /W3
#pragma warning(default : 4287)
#include <stdio.h>

int main()
{
    unsigned int u = 1;
    if (u < -1)   // C4287
        printf_s("u LT -1");
    else
        printf_s("u !LT -1");
    return 0;
}
```
