# <a name="compiler-warning-level-1-c4045"></a>Advertencia del compilador (nivel 1) C4045
'array': desbordamiento de límites de matriz  
  
 La matriz tiene hay demasiados inicializadores. Los inicializadores adicionales se omiten.  
  
 Asegúrese de que elementos de matriz y los inicializadores coinciden en tamaño y cantidad.
