# <a name="compiler-warning-level-1-c4154"></a>Compilador advertencia (nivel 1) C4154
eliminación de una expresión de matriz; conversión de puntero proporcionado  
  
 No se puede utilizar `delete` en una matriz, por lo que el compilador convierte la matriz en un puntero.  
  
## <a name="example"></a>Ejemplo  
  
```  
// C4154.cpp  
// compile with: /c /W1  
int main() {  
   int array[ 10 ];  
   delete array;   // C4154 can't delete stack object  
  
   int *parray2 = new int [10];  
   int (&array2)[10] = (int(&)[10]) parray2;  
   delete [] array2;   // C4154  
  
   // try the following line instead  
   delete [] &array2;  
}  
```
