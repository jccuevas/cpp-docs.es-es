# <a name="compiler-warning-level-1-c4375"></a>Advertencia del compilador (nivel 1) C4375
el método no público 'method2' no reemplazar 'method1'  
  
 Un tipo que implementa otro tipo definido por un método de reemplazo, pero el reemplazo no fue público. Por lo tanto, el método no reemplaza el método de tipo base.
