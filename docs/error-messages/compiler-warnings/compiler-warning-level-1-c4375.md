# <a name="compiler-warning-level-1-c4375"></a>Advertencia del compilador (nivel 1) C4375

el método no público 'method2' no invalida 'method1'

Define un tipo que implementa el otro tipo de un método de invalidación, pero el reemplazo no es público. Por lo tanto, el método no invalida el método de tipo base.
