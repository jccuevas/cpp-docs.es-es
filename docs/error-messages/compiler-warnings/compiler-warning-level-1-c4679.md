# <a name="compiler-warning-level-1-c4679"></a>Advertencia del compilador (nivel 1) C4679

> '*member*': no se pudo importar el miembro

El compilador encontró una construcción que no admite, que no se puede importar desde los metadatos.

No intente usar la construcción.
