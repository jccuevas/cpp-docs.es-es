# <a name="compiler-warning-level-3-c4724"></a>Advertencia del compilador (nivel 3) C4724
posible división (módulo) por 0  
  
 El segundo operando en una operación de resto que se evalúa como cero en tiempo de compilación, que da resultados no definidos.
