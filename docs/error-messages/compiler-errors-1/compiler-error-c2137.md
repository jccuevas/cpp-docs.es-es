# <a name="compiler-error-c2137"></a>Error del compilador C2137
constante de caracteres vacía  
  
 La constante de carácter vacío (' ') no está permitida.  
  
 El ejemplo siguiente genera la advertencia C2137:  
  
```  
// C2137.cpp  
int main() {  
   char c = '';   // C2137  
   char d = ' ';   // OK  
}  
```
