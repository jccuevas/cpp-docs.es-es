# Error del compilador C2168
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'función': no hay suficientes parámetros reales para la función intrínseca  
  
 Una referencia a una función `intrinsic` tiene demasiado pocos parámetros.
