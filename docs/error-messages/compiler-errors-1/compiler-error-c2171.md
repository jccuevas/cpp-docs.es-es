# <a name="compiler-error-c2171"></a>Error del compilador C2171
'operator': no es válido para operandos de tipo 'type'  
  
 Se usa un operador unario con un tipo de operando no válido.  
  
## <a name="example"></a>Ejemplo  
 El ejemplo siguiente genera la advertencia C2171.  
  
```  
// C2171.cpp  
int main() {  
   double d, d1;  
   d = ~d1;   // C2171  
  
   // OK  
   int d2 = 0, d3 = 0;  
   d2 = ~d3;  
}  
```  
  
## <a name="example"></a>Ejemplo  
 El ejemplo siguiente genera la advertencia C2171.  
  
```  
// C2171_b.cpp  
// compile with: /c  
class A {  
public:  
   A() { STF( &A::D ); }  
  
   void D() {}  
   void DTF() {  
      (*TF)();   // C2171  
      (this->*TF)();   // OK  
   }  
  
   void STF(void (A::*fnc)()) {  
      TF = fnc;  
   }  
  
private:  
   void (A::*TF)();  
};  
```
