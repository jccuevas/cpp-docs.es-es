# Error irrecuperable C1057
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

final de archivo inesperado en la expansión de macros  
  
 El compilador ha alcanzado el final del archivo de código fuente mientras recopilaba argumentos de invocación de macros, probablemente debido a la falta de algún paréntesis en la invocación de macro.
