# Error del compilador C2028
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

el miembro struct\/union debe estar dentro de una estructura o unión  
  
 Los miembros de estructura o de unión deben declararse dentro de la estructura o la unión.
