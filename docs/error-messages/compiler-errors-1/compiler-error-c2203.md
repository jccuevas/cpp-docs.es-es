# <a name="compiler-error-c2203"></a>Error del compilador C2203

Eliminar operador no puede especificar límites de matriz

Con el **/Za** opción (ANSI), el `delete` operadores pueden eliminar toda una matriz, pero no elementos o miembros específicos de la matriz.

El ejemplo siguiente genera C2203:

```
// C2203.cpp
// compile with: /Za
int main() {
   int *ar = new int[10];
   delete [4] ar;   // C2203
   // try the following line instead
   // delete [] ar;
}
```
