# Error del compilador C2311
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'excepción' : ha sido detectada por '...' en la línea número  
  
 El controlador catch de puntos suspensivos \(...\) debe ser el último controlador para throw.  
  
 El código siguiente genera el error C2311:  
  
```  
// C2311.cpp  
// compile with: /EHsc  
#include <eh.h>  
int main() {  
   try {  
      throw "ooops!";  
   }  
   catch( ... ) {}  
   catch( int ) {}   // C2311  ellipsis handler not last catch  
}  
```
