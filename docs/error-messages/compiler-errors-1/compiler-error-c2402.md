# Error del compilador C2402
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'identificador': el registro debe ser de índice en 'contexto'.  
  
 El registro usado en un operando de memoria indirecta debe ser un registro de índice en este contexto.
