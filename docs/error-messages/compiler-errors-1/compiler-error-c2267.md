# <a name="compiler-error-c2267"></a>Error del compilador C2267
'función': las funciones static con ámbito de bloque no son válidas  
  
 Se declara una función local `static`. Funciones estáticas deben tener ámbito global.  
  
 El ejemplo siguiente genera C2267:  
  
```  
// C2267.cpp  
static int func2();   // OK  
int main() {  
    static int func1();   // C2267  
}  
```
