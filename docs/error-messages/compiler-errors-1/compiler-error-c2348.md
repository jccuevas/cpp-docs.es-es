# <a name="compiler-error-c2348"></a>Error del compilador C2348

'type name': no es un agregado de estilo C, no se puede exportar en IDL incrustado

Para colocar un `struct` en un archivo IDL con la [exportar](../../windows/export.md) atributo, el `struct` s√≥lo debe contener datos.

El ejemplo siguiente genera C2348:

```
// C2348.cpp
// C2348 error expected
[ module(name="SimpleMidlTest") ];

[export]
struct Point {
   // Delete the following two lines to resolve.
   Point() : m_i(0), m_j(0) {}
   Point(int i, int j) : m_i(i), m_j(j) {}

   int m_i;
   int m_j;
};
```
