# <a name="compiler-error-c2109"></a>Error del compilador C2109

el subíndice requiere el tipo de puntero o matriz

El subíndice se usó en una variable que no era una matriz.

El ejemplo siguiente genera C2109:

```
// C2109.cpp
int main() {
   int a, b[10] = {0};
   a[0] = 1;   // C2109
   b[0] = 1;   // OK
}
```
