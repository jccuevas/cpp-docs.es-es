# Error irrecuperable del compilador de recursos RC1021
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

comando de preprocesador 'cadena' no válido  
  
 Los caracteres que siguen al signo de número \(**\#**\) no formaron una directiva de preprocesador válida.  
  
 El signo de número no se puede utilizar como el primer carácter en un identificador.
