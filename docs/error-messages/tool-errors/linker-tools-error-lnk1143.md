# Error de las herramientas del vinculador LNK1143
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

archivo no válido o dañado: no hay símbolo para la sección 'número' de COMDAT  
  
 Este error puede producirse si el archivo del objeto está dañado.  
  
 Recompile el archivo.
