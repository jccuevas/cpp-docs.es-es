# <a name="resource-compiler-error-rc2165"></a>Error del compilador de recursos RC2165

cadena que no se encuentra en la instrucción DLGINCLUDE

La instrucción no especificó un archivo de inclusión válida.

El **DLGINCLUDE** instrucción debe usar la sintaxis siguiente:

**DLGINCLUDE "*filename*. h"**
