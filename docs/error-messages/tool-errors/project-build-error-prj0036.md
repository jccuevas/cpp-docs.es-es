# <a name="project-build-error-prj0036"></a>Error PRJ0036 al compilar el proyecto

La propiedad 'Archivos adicionales' de la herramienta de implementación Web contenía una entrada no válida.

La propiedad de archivos adicionales en la página de propiedades de implementación Web contenía un error, posiblemente debido a un problema de evaluación de macro. Este error también puede significar que la ruta de acceso es incorrecto, que contiene caracteres o combinaciones de caracteres que no son válidos en una ruta de acceso de archivo.

Para resolver este error, corrija la macro o corrija la especificación de ruta de acceso. La ruta de acceso evaluada es una ruta de acceso absoluta del directorio de proyecto.

Este error también puede significar que uno de los archivos al que hace referencia no existe.
