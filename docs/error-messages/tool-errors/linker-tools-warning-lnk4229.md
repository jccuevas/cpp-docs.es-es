# <a name="linker-tools-warning-lnk4229"></a>Advertencia de las herramientas del vinculador LNK4229

se encontró una directiva no válida/Directive; tendrán

Una directiva que se pasa a través de la pragma [comment](../../preprocessor/comment-c-cpp.md) no era válida. El enlazador omite `/directive`.
