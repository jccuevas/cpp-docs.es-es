# <a name="expression-evaluator-error-cxx0066"></a>Error del evaluador de expresiones CXX0066
miembro estático no está presente  
  
 Un miembro estático de una clase no se pudo encontrar o no se ha definido. Este error puede deberse a desde una clase estática miembro que se ha declarado pero no definido, o es solo definido y se hace referencia en módulos que no contienen información de depuración.  
  
 Este error es idéntico a CAN0066.
