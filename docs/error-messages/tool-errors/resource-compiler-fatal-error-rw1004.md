# <a name="resource-compiler-fatal-error-rw1004"></a>Error irrecuperable del compilador de recursos RW1004

Final de archivo inesperado

Este error puede deberse a que faltan caracteres de salto de línea y de retorno de carro en la última línea de un archivo de texto.
