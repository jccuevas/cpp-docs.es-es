# Error irrecuperable del compilador de recursos RC1009
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

límite del compilador : las macros están demasiado anidadas 'macro'  
  
 El archivo supera el límite del compilador de recursos en el anidamiento de macros.  Revise el archivo de código fuente especificado para disminuir el nivel de anidamiento de las macros.
