# <a name="resource-compiler-error-rc2112"></a>Error del compilador de recursos RC2112
Se esperaba BEGIN en el cuadro de diálogo  
  
 El **comenzar** palabra clave debe seguir inmediatamente el **diálogo** palabra clave.
