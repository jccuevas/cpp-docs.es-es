# <a name="bscmake-error-bk1512"></a>Error de BSCMAKE BK1512

nombre de archivo: se excedió la capacidad

BSCMAKE no puede generar un archivo de información de examen porque el número de definiciones, referencias, módulos u otra información supera el límite.

### <a name="to-fix-by-using-the-following-possible-solutions"></a>Use las soluciones posibles siguientes para corregirlo

1. Excluya cierta información mediante /Em, / es o/EI.

1. Omite la opción/IU.
