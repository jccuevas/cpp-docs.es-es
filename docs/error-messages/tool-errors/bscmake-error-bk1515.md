# <a name="bscmake-error-bk1515"></a>Error de BSCMAKE BK1515

archivo_bsc: versión incompatible: no se puede actualizar incrementalmente

Se creó el archivo .bsc con otra versión de BSCMAKE. Vuelva a crear el archivo .bsc.
