# <a name="command-line-warning-d9024"></a>Advertencia de la línea de comandos D9024

origen no reconocido 'filename' de tipo de archivo, se supone el archivo objeto

No se reconoció la extensión del archivo especificado. El archivo se supone que un archivo de objeto y se pasó al vinculador.

Se reconocen las siguientes extensiones:

- .c (archivo de código fuente de C)

- .cxx (archivo de código fuente de C++)

- .cpp (archivo de código fuente de C++)

- .obj (archivo objeto)

- .lib (archivo de biblioteca)

- .def (archivos de definición de módulo)

- .exp (archivo, creado por/lib de vínculo de exportaciones del vinculador)
