# <a name="compiler-error-c3832"></a>Error del compilador C3832
'biblioteca de tipos': parece que la biblioteca de tipos se compiló para punteros de 32 bits; Cambie el calificador 'ptrsize'  
  
 Proporcionó información explícita con el `ptrsize` atributo de la [#import](../../preprocessor/hash-import-directive-cpp.md) directiva no está de acuerdo con lo que el compilador se encuentra en la biblioteca de tipos.
