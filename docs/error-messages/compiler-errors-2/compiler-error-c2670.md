# <a name="compiler-error-c2670"></a>Error del compilador C2670

'identifier': la plantilla de función no puede convertir el número de parámetro de tipo 'type'

Un parámetro no se pudo convertir al tipo requerido.

Este error puede corregirse si creas una conversión explícita.
