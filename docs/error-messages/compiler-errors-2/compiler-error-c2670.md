# Error del compilador C2670
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'identificador' : la plantilla de función no puede convertir el parámetro número desde el tipo 'tipo'  
  
 No se puede convertir un parámetro al tipo requerido.  
  
 Podrá corregir este error si crea una conversión explícita.
