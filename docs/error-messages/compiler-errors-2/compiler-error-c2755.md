# Error del compilador C2755
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

'param' : el parámetro sin tipo de una especialización parcial debe ser un identificador simple  
  
 El parámetro sin tipo debe ser un identificador simple para que el compilador pueda resolverlo en tiempo de ejecución en un único identificador o valor constante.  
  
 El código siguiente genera el error C2755:  
  
```  
// C2755.cpp  
template<int I, int J>  
struct A {};  
  
template<int I>   
struct A<I,I*5> {};   // C2755  
// try the following line instead  
// struct A<I,5> {};  
```
