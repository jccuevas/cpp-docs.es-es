# <a name="compiler-error-c2568"></a>Error del compilador C2568
'identificador1': no se puede resolver la sobrecarga de función  
  
 El compilador no puede determinar qué función llamar a sobrecargada. Los parámetros reales pasados a la función se deben convertir para que coincida con los parámetros formales de una función sobrecargada, pero no hay una coincidencia es inequívoca mejor que todos los demás.
