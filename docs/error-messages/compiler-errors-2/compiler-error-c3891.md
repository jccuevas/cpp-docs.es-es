# <a name="compiler-error-c3891"></a>Error del compilador C3891

'var': un miembro de datos literal no puede usarse como un valor l

Un [literal](../../windows/literal-cpp-component-extensions.md) variable es const, y su valor no puede cambiarse una vez que se inicializa en la declaraci√≥n.

El ejemplo siguiente genera C3891:

```
// C3891.cpp
// compile with: /clr
ref struct Y1 {
   literal int staticConst = 9;
};

int main() {
   Y1::staticConst = 0;   // C3891
}
```
