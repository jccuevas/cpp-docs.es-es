# Error del compilador C2858
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

opción de la línea de comandos '\/Yc \(\/Fdfilename\)' incoherente con el encabezado precompilado, que utilizó '\/Fdfilename'  
  
 La base de datos de programa especificada por la opción Utilizar encabezado precompilado \([\/Yu](../../build/reference/yu-use-precompiled-header-file.md)\) no es la misma que se especificó en la opción anterior Crear encabezado precompilado \([\/Yc](../../build/reference/yc-create-precompiled-header-file.md)\).
