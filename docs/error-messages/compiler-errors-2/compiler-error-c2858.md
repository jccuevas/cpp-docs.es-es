# <a name="compiler-error-c2858"></a>Error del compilador C2858

la opción de línea de comandos '/YC (/Fdfilename) ' es incoherente con el encabezado precompilado, que usaba '/Fdfilename '

La base de datos de programa especificada por la opción usar encabezado precompilado ([/Yu](../../build/reference/yu-use-precompiled-header-file.md)) no es la especificada por la opción anterior crear encabezado precompilado ([/YC](../../build/reference/yc-create-precompiled-header-file.md)).
