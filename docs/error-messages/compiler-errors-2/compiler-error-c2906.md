# Error del compilador C2906
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

“especialización”: la especialización explícita requiere la “plantilla \<\>”  
  
 Debe utilizar la nueva sintaxis para la especialización explícita de plantillas.  
  
 El código siguiente genera el error C2906:  
  
```  
// C2906.cpp  
// compile with: /c  
template<class T> class X{};   // primary template  
class X<int> { }   // C2906  
template<> class X<int> { };   // new syntax  
```
