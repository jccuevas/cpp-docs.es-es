# Error del compilador C2543
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

se esperaba '\]' para el operador '\[\]'  
  
 Al operador de subíndice le falta un corchete izquierdo.  
  
 Este error puede estar motivado por la expansión de macro.
