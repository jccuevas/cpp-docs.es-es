# <a name="compiler-error-c3340"></a>Error del compilador C3340
'interfaz': la interfaz no puede ser 'restricted' y 'default' en la coclase 'class'.  
  
 El [restringido](../../windows/restricted.md) atributo y el [predeterminado](../../windows/default-cpp.md) atributo son mutuamente excluyentes.  
  
 El ejemplo siguiente genera la advertencia C3340:  
  
```  
// C3340.cpp  
#include <windows.h>  
[module(name="MyModule")];  
  
[ object, uuid(373a1a4c-469b-11d3-a6b0-00c04f79ae8f) ]  
__interface IMyIface  
{  
   HRESULT f1();  
};  
  
[ coclass, uuid(373a1a4d-469b-11d3-a6b0-00c04f79ae8f),  
default(IMyIface),  
source(IMyIface),restricted(IMyIface) ]  
class CmyClass // C3340  
{  
};  
  
int main()  
{  
}  
```
