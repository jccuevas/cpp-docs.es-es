# iter_swap (STL/CLR)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

Los intercambios dos valores se refirieron por un par de iteradores especificados.  
  
## Sintaxis  
  
```  
template<class _FwdIt1, class _FwdIt2> inline  
    void iter_swap(_FwdIt1 _Left, _FwdIt2 _Right);  
```  
  
## Comentarios  
 Esta función se comporta igual que la función `iter_swap`STL.  Para obtener más información, vea [iter\_swap](../Topic/iter_swap.md).
