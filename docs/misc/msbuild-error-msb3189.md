# Error de MSBuild MSB3189
**MSB3189: La ubicación de actualizaciones para esta aplicación es una ruta de acceso local.**  
  
 Esta advertencia sólo se aplica a los manifiestos de implementación.  
  
### Para corregir este error  
  
-   Asegúrese de que la ubicación de actualización es una dirección URL válida.
