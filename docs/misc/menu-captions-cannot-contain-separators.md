# Menu captions cannot contain separators.
Este error ocurre normalmente cuando se usa el carácter de guión \(\-\) en el nombre de una leyenda de menú.  El carácter de guión está designado para los separadores de menú.  
  
### Para corregir este error  
  
1.  Escriba otro nombre para la leyenda que no utilice el guión.
