# A failure occurred while attempting to start the compilation
Se encontró un error desconocido al emitir la solicitud **Iniciar compilación** para el compilador de [!INCLUDE[vbprvb](../dotnet/includes/vbprvb_md.md)] o de [!INCLUDE[csprcs](../ide/includes/csprcs_md.md)].  
  
 **Para corregir este error**  
  
-   Reinicie [!INCLUDE[vsprvs](../assembler/masm/includes/vsprvs_md.md)].  
  
     Esto provocará un error en el proceso de compilación.
