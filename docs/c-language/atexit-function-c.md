# atexit (Funci&#243;n) (C)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**ANSI 4.10.4.3** El estado devuelto por la función `atexit` si el valor del argumento es distinto de cero, EXIT\_SUCCESS o EXIT\_FAILURE  
  
 La función `atexit` devuelve cero si es correcta, o un valor distinto de cero si es incorrecta.
