# fmod (Funci&#243;n)
[!INCLUDE[vs2017banner](../assembler/inline/includes/vs2017banner.md)]

**ANSI 4.5.6.4** Indica si se produce un error de dominio o se devuelve cero cuando la función `fmod` tiene un segundo argumento de cero  
  
 Cuando la función `fmod` tiene un segundo argumento de cero, la función devuelve cero.
