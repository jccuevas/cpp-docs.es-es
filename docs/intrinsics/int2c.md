# <a name="int2c"></a>__int2c
**Específicos de Microsoft**  
  
 Genera el `int 2c` instrucción, lo que desencadena la `2c` interrupción.  
  
## <a name="syntax"></a>Sintaxis  
  
```  
void __int2c(void);  
```
