## <a name="swap"></a>  swap

Intercambia los elementos de dos deques.

```cpp
void swap(
    deque<Type, Allocator>& left,
    deque<Type, Allocator>& right,);
```

### <a name="parameters"></a>Parámetros

*izquierdo* un objeto de tipo `deque`.

*derecha* un objeto de tipo `deque`.
