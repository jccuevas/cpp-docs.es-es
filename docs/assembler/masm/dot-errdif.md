# .ERRDIF]]
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Genera un error si los elementos de texto son diferentes.  
  
## Sintaxis  
  
```  
  
.ERRDIF[[I]] <textitem1>, <textitem2> [[, message]]  
```  
  
## Comentarios  
 Si se especifica **i** , la comparación no distingue entre mayúsculas y minúsculas.
