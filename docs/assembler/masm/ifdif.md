# IFDIF]
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Concede al ensamblado si los elementos de texto son diferentes.  
  
## Sintaxis  
  
```  
  
IFDIF[[I]] textitem1, textitem2  
```  
  
## Comentarios  
 Si se especifica **i** , la comparación no distingue entre mayúsculas y minúsculas.  Vea [IF &#91;SQL2008&#93;](../../assembler/masm/if-masm.md) para la sintaxis completa.
