# operator LROFFSET
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Devuelve la diferencia *de la expresión*.  Igual que **Desplazamiento**, pero genera un desplazamiento resuelto cargador, que permite que Windows reubique segmentos de código.  
  
## Sintaxis  
  
```  
  
LROFFSET expression  
```
