# operator OPATTR
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

devuelve una palabra que define el modo y el ámbito de la *expresión*.  El byte bajo es idéntico al byte devuelto por [.TYPE](../../assembler/masm/operator-dot-type.md).  El alto byte contiene información adicional.  
  
## Sintaxis  
  
```  
  
OPATTR expression  
```
