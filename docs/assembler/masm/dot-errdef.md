# <a name="errdef"></a>.ERRDEF
Genera un error si *nombre* es una etiqueta definida previamente, la variable o el símbolo.  
  
## <a name="syntax"></a>Sintaxis  
  
```  
  
.ERRDEF name [[, message]]  
```
