# <a name="db"></a>DB
Asigna y, opcionalmente, inicializa un byte de almacenamiento para cada `initializer`. `DB` es un sinónimo de [bytes](../../assembler/masm/byte-masm.md)  
  
## <a name="syntax"></a>Sintaxis  
  
```  
[[name]] DB initializer [[, initializer]] ...  
```  
  
## <a name="remarks"></a>Comentarios  
 También puede usarse como un especificador de tipo en cualquier lugar un tipo es legal.
