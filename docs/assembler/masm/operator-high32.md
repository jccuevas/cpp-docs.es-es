# <a name="operator-high32"></a>Operador HIGH32
Devuelve los 32 bits superiores de *expresión*. Expresiones MASM son valores de 64 bits.  
  
## <a name="syntax"></a>Sintaxis  
  
```  
  
HIGH32 expression  
```
