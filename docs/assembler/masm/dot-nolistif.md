# .NOLISTIF
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Suprime la lista de bloques condicionales cuya condición devuelva false \(0\).  
  
## Sintaxis  
  
```  
  
.NOLISTIF  
  
```  
  
## Comentarios  
 Éste es el valor predeterminado.  Igual que [.SFCOND](../../assembler/masm/dot-sfcond.md).
