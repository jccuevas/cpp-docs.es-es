# operator MOD
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Devuelve el valor entero del resto \(módulo\) al dividir *expression1* por *expression2*.  
  
## Sintaxis  
  
```  
  
expression1 MOD expression2  
```
