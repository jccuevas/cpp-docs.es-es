# SQWORD
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Asigna e inicializa opcionalmente 8 bytes firmados de almacenamiento para cada `initializer`.  También se puede utilizar como especificador de tipo siempre que un tipo es válido.  
  
## Sintaxis  
  
```  
  
name SQWORD initializer [[, initializer]]...  
```
