# <a name="const"></a>.CONST

Cuando se usa con [. MODELO](../../assembler/masm/dot-model.md), comienza un segmento de datos constantes (por segmento de nombre CONST).

## <a name="syntax"></a>Sintaxis

> .CONST

## <a name="remarks"></a>Comentarios

Este segmento tiene el atributo de sólo lectura.
