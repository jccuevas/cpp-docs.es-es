# <a name="operator-lt-masm-run-time"></a>operador &lt;= (tiempo de ejecución de MASM)

Es menor o igual que. Se usa solo dentro de [. Si](../../assembler/masm/dot-if.md)es, [. WHILE](../../assembler/masm/dot-while.md), o [. REPETIR](../../assembler/masm/dot-repeat.md) bloques y evaluarlos en tiempo de ejecución, no en tiempo de ensamblado.

## <a name="syntax"></a>Sintaxis

> *expression1* **<=** *expression2*
