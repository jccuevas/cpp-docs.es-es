# <a name="operator-ne"></a>Operador NE
Devuelve true (-1) si *expression1* no es igual a *expression2*, o devuelve false (0) si lo hace.  
  
## <a name="syntax"></a>Sintaxis  
  
```  
  
expression1 NE expression2  
```
