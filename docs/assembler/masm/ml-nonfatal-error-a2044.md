# ML Nonfatal Error A2044
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

**carácter no válido en archivo**  
  
 El archivo de código fuente contiene un carácter fuera de un comentario, una cadena, o un literal que no ha reconocido como operador u otro carácter válido.
