# .LISTMACRO
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Inicia la lista de instrucciones de expansión de macro que generan código o datos.  
  
## Sintaxis  
  
```  
  
.LISTMACRO  
  
```  
  
## Comentarios  
 Éste es el valor predeterminado.  Igual que [.XALL](../../assembler/masm/dot-xall.md).
