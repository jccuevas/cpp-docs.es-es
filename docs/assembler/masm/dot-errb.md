# .ERRB
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Genera un error si *el textitem* está en blanco.  
  
## Sintaxis  
  
```  
  
.ERRB <textitem> [[, message]]  
```
