# PUBLIC (MASM)
[!INCLUDE[vs2017banner](../../assembler/inline/includes/vs2017banner.md)]

Crea cada variable, etiqueta, o símbolo absoluto especificado como *nombre* disponible al resto de los módulos del programa.  
  
## Sintaxis  
  
```  
  
PUBLIC [[langtype]] name [[, [[langtype]] name]]...  
```
