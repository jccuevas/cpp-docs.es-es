# <a name="c26448-use_gsl_finally"></a>USE_GSL_FINALLY C26448

Considere la posibilidad de usar `gsl::finally` si la acción final está prevista.

C++Directrices básicas: [GSL. util: Utilidades](https://github.com/isocpp/CppCoreGuidelines/blob/master/CppCoreGuidelines.md#SS-utilities)

La biblioteca de compatibilidad de directrices proporciona una utilidad adecuada para implementar el concepto de *acción final* . Dado que C++ el lenguaje no admite construcciones **try-finally** , se convirtió en común implementar tipos de limpieza personalizados que invocarían acciones arbitrarias en la destrucción. La utilidad de `gsl::finally` se implementa de esta manera y proporciona una manera más uniforme de realizar las acciones finales en un código base.

También hay casos en los que se realizan acciones finales en una forma de estilo C de uso antiguo mediante instrucciones **goto** (que por lo general no se recomienda en [C26438 NO_GOTO](c26438.md)). Es difícil detectar la intención exacta en el código que usa mucho **goto**, pero algunas heurísticas pueden ayudar a buscar mejores candidatos para la limpieza.

## <a name="remarks"></a>Observaciones

- Esta regla es muy ligera y usa nombres de etiqueta para adivinar las oportunidades de usar objetos de acción finales.
- Los nombres de etiqueta que pueden generar una advertencia contienen palabras como "final", "final", "Clean", etc.
- Las advertencias aparecen en las instrucciones **goto** . Puede ver resultados detallados en algunas ocasiones, pero esto puede ayudarle a priorizar el código en función de su complejidad.
- Esta regla siempre entra en pareja con [C26438 NO_GOTO](c26438.md). En función de las prioridades, una de ellas puede deshabilitarse.

## <a name="example-cleanup-with-multiple-goto-statements"></a>Ejemplo: limpieza con varias instrucciones Goto

```cpp
void poll(connection_info info)
{
    connection c = {};
    if (!c.open(info))
        return;

    while (c.wait())
    {
        connection::header h{};
        connection::signature s{};
        if (!c.read_header(h))
            goto end;               // C26448 and C26438
        if (!c.read_signature(s))
            goto end;               // C26448 and C26438
        // ...
    }

end:
    c.close();
}
```

## <a name="example-cleanup-with-multiple-goto-statements-replaced-by-gslfinally"></a>Ejemplo: limpieza con varias instrucciones Goto reemplazadas por GSL:: Finally

```cpp
void poll(connection_info info)
{
    connection c = {};
    if (!c.open(info))
        return;

    auto end = gsl::finally([&c] { c.close(); });
    while (c.wait())
    {
        connection::header h{};
        connection::signature s{};
        if (!c.read_header(h))
            return;
        if (!c.read_signature(s))
            return;
        // ...
    }
}
```
